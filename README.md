# Astro Getting Started 
## Deploying On Liara
- create a static app in [Liara](https://console.liara.ir/apps/create)
```
git clone https://github.com/liara-cloud/astro-getting-started.git
```
```
cd astro-getting-started
```
```
npm run install
```
```
npm run build
```
```
cd dist
```
```
liara deploy 
```
